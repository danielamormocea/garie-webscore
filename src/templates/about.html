This site provides software quality checks and ratings for EEA websites. Hundreds of quality checks are performed daily and results can be visualized in a single unified dashboard.

The rating scale is from 0 (worst) to 100 (best).

The ratings evolution over time for each EEA site can also be visualized, with details for each check being available.

Checks and tests can be easily added by creating plugins (starting from an existing template); plugins can also collect results from other external systems.

For more info please visit The Garie wiki page.
